# MINST
將所有檔案放在同個資料夾

使用VSCODE開啟資料夾

在終端機輸入
```
npx serve
``` 
將Local的地址輸入瀏覽器中

![image](https://user-images.githubusercontent.com/22973216/210082700-afa718c4-6ce0-4539-acbb-fccd2a682ee3.png)

在畫布內寫上數字

打開瀏覽器主控台查看結果
